### ⚾️ 숫자 야구
<br>

> 0 ~ 9까지 무작위의 번호를 생성한다.
> - ✅ 중복이 존재하지 않는가?
> - ✅ round 만큼 숫자가 생성되는가?
> - ✅ 새로고침했을때마다 새로운 번호가 생성되는가?

> 사용자가 번호를 클릭했을 때
> - ✅ 숫자가 존재하면서 자릿수(위치)까지 맞으면 스트라이크 카운트가 올라가는가?
> - ✅ 숫자가 존재하지만 자릿수(위치)까지 다르면 볼 카운트가 올라가는가?
> - ✅ 0 스트라이크 0 볼인 숫자들은 다시 클릭할 수 없도록 진행되는가?

> 특정 라운드 클리어했을 때
> - ✅ 해마의 크기가 변하는가?
> - ✅ 최종 라운드 클리어시 게임 종료를 알리는가?
